# Loup Solitaire (LDVELH) - Web App

Une application web moderne pour gérer vos aventures "Loup Solitaire" (Livre Dont Vous Êtes Le Héros).

## Fonctionnalités

- **Feuille d'Aventure Complète** : Gérez votre Habileté, Endurance et Chance.
- **Gestion d'Équipement** : Suivez votre or, vos repas et vos objets.
- **Maîtrise des Armes** : Système de progression pour vos armes.
- **Journal de Bord** : Historique détaillé de vos chapitres visités avec notes et tags.
- **Multiprofil** : Gérez plusieurs aventures simultanément.
- **Combat Immersif** : Interface dédiée pour les combats avec jets de dés animés.

## Technologies

- **React** avec **Vite**
- **Lucide-React** (Icones)
- **Framer Motion** (Animations)
- **Vanilla CSS** pour l'esthétique "Grimoire"

## Installation

1. Clonez le dépôt
2. Installez les dépendances :

   ```bash
   npm install
   ```

3. Lancez le serveur de développement :

   ```bash
   npm run dev
   ```
